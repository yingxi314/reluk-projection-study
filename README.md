# ReLU^k Projection Study

This repository contains reading notes and small-scale numerical reproductions related to L2 and H1 projections in ReLU^k neural spaces.

The work is based on several papers on linearized neural networks, ReLU^k activation functions, and approximation methods for PDE-related problems. The goal is not to propose a new method, but to better understand the numerical formulation, including weighted least-squares problems, matrix assembly, conditioning, and the comparison of different integration strategies such as quadrature, Monte Carlo / Sobol-QMC sampling, and polynomial integration.
